Автоматический режим.
ATmega16

Есть две ветки меню:
 1-я - настройки программы,
 2-я - общие настройки.
В меню программы для каждой программы настраивается режим сварки и, если он автоматический, настраивается пауза между циклами сварки.
В общих настройках настраивается номер программы, с которой будет стартовать БУ
